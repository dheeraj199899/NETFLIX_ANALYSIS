# 📊 Netflix Movies & TV Shows Dashboard | End-to-End Data Analytics Project

## 📌 Project Overview

This project demonstrates an end-to-end **Data Analytics** workflow using **Microsoft Excel, PostgreSQL, SQL, Power Query, DAX, and Power BI**. Starting with a raw Netflix Movies & TV Shows dataset downloaded from Kaggle, I cleaned, transformed, analyzed, and visualized the data to create an interactive dashboard that provides meaningful insights into Netflix's content library.

The objective of this project was not only to build a dashboard but also to simulate a real-world data analytics process by working with raw data, performing data cleaning at multiple stages, writing SQL queries for analysis, and creating meaningful business insights.

---

## 🚀 Project Workflow

### 1️⃣ Data Collection

- Downloaded the raw Netflix Movies & TV Shows dataset from Kaggle.
- Explored the dataset to understand its structure, columns, missing values, and data quality issues.

---

### 2️⃣ Data Cleaning in Microsoft Excel

Performed initial data cleaning by:

- Removing unnecessary spaces
- Fixing inconsistent formatting
- Handling missing values
- Standardizing text values
- Preparing the dataset for SQL import

---

### 3️⃣ Data Processing Using PostgreSQL

Imported the cleaned dataset into PostgreSQL and performed additional data cleaning and analysis using SQL.

**Tasks Performed**

- Updated incorrect column values
- Handled missing values
- Cleaned inconsistent records
- Standardized data
- Wrote SQL queries to answer business-related questions
- Explored trends and validated data quality

---

### 4️⃣ Data Transformation in Power Query

After processing the data in PostgreSQL, I exported it as a CSV file and imported it into Power BI.

Using **Power Query**, I:

- Checked data types
- Removed duplicate records
- Replaced hyphens with spaces for better readability
- Verified data consistency
- Applied final transformations before loading the data model

---

### 5️⃣ Data Modeling & DAX

Created DAX measures to calculate key business metrics, including:

- 📌 Total Titles
- ⭐ Average Rating
- 🔥 Average Popularity
- 👍 Total Votes

These measures were used to build dynamic and interactive visualizations.

---

### 6️⃣ Dashboard Development

Designed an interactive Power BI dashboard featuring:

- KPI Cards
- Release Year Trend Analysis
- Genre-wise Popularity Analysis
- Genre-wise Average Rating
- Popularity vs Rating Scatter Plot
- Interactive Filters / Slicers
- Movies & TV Shows Detail Table

The dashboard allows users to explore Netflix content dynamically through interactive filters and visualizations.

---

# 📈 Business Insights

The dashboard helps answer questions such as:

- 📌 How has Netflix content grown over the years?
- 🎬 Which genres are the most popular?
- ⭐ Which genres have the highest average ratings?
- 📊 Is there a relationship between popularity and ratings?
- 📅 How are titles distributed across release years?
- 👍 Which titles receive the highest audience engagement based on votes?
